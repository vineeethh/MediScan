# Screenshots Directory

Place your project screenshots here with the following naming:

1. **landing.png** - Landing page with hero section
2. **chat.png** - AI Doctor chat interface
3. **hospitals.png** - Hospital finder with map
4. **dashboard.png** - User dashboard
5. **dark-mode.png** - Dark mode interface
6. **mobile.png** - Mobile responsive view

## How to Take Screenshots

### Mac
- Cmd + Shift + 4 (Select area)
- Cmd + Shift + 3 (Full screen)

### Windows
- Windows + Shift + S (Snipping tool)
- PrtScn (Full screen)

## Tips for Good Screenshots

- Use a clean browser window (no tabs clutter)
- Make sure no personal information is visible
- Use test data, not real user data
- Capture key features clearly
- Use consistent window size
- Consider using sample text for better demo

## Image Optimization

Before committing, optimize images to reduce size:

```bash
# Install ImageOptim (Mac)
brew install imageoptim-cli

# Optimize all screenshots
imageoptim screenshots/*.png
```

Or use online tools:
- [TinyPNG](https://tinypng.com/)
- [Squoosh](https://squoosh.app/)
